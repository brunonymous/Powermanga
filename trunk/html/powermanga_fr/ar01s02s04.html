2.4. Les commandes pendant le jeu.

Touches fléchées : utilisées pour déplacer le vaisseau.

ESPACE : pour déclancher les tirs du vaisseau.

CTRL : achète l’option active.

ECHAP : affiche le menu (un second appui sur ECHAP pour revenir au jeu)

P : met le jeu en pause (un second appui sur P Permet de reprendre le jeu)

F : permet de passer du mode plein écran au mode fenêtré (et vice versa).
